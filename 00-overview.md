# HTML Table

## What I Learned
- The tag '<table></table>'  is used to create a table within the webpage
- The tag '<tr>' is used to create a row
- The tag '<td>' creates cells within the row

## Example
```html
<table>
        <tr>
            <th>Date</th>
            <th>Meter reading</th>
        </tr>
        <tr>
            <td>21/06/2025</td>
            <td>5375.911</td>
        </tr>
    </table>
